# Library Management System (LMS)

This is a simple Library Management System made using HTML, CSS, and JavaScript.
The project simulates basic library operations like adding books, adding students, issuing books, returning books, and admin login/register.

## Features

* Admin Login and Register
* Dashboard with navigation
* Add Book form
* Add Student form
* Issue Book
* Return Book
* Issued Books table update
* Background library image UI
* Popup success messages
* Pure frontend project (no database)

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Download or clone the repository
2. Make sure all files are in the same folder:

   * index.html
   * style.css
   * library.jpg
3. Open index.html in browser
